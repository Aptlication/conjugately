# French Verb Master — Project Synopsis

**Site:** [frenchverbmaster.com](https://frenchverbmaster.com/)
**Tagline:** *"For serious students"*

## Purpose

A French language learning web app focused specifically on verb conjugation mastery — positioned as the core skill required for fluency.

## Core Feature: Personalised Conjugation Quizzes

Users select three parameters to generate a tailored quiz:

1. **Difficulty level** — from Beginner through to Intermediate (Advanced coming soon)
2. **French verb** — drawn from a curated verb list per difficulty tier
3. **Time frame** — Present, Past, or Future tense

## Difficulty Tiers

- ⚪ **Beginner** — Subject pronoun focus (*Je suis, Tu es*), 3 verbs (*être, avoir, faire*)
- 🔵 **Novice** — 4 verbs including *aller*; 3 tenses
- 🟢 **Elementary** — 7 verbs; 3 tenses
- 🟡 **Intermediate** — 18 verbs (7 reflexive + 11 non-reflexive); 3 tenses
- 🔒 **Advanced** — Coming soon

## Secondary Features

- 📚 **Mini-Courses** — structured learning modules
- 📖 **Vocabulary** — supplementary vocab section
- 🎲 **"Choose All for Me"** — randomised quiz shortcut

## Technical / Business Notes

- Standalone web app with clean single-page UI
- Includes a Privacy Policy page
- Positioned for serious/motivated learners, not casual apps like Duolingo
- Potential monetisation or upsell via mini-courses
