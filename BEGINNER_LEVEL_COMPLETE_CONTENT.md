## COMPLETE QUIZ TEMPLATES

### QUIZ TEMPLATE: ÊTRE + PRÉSENT (Present Tense)

**Question 1**: "I am happy."
- **Hint**: Use the present tense form of être for je
- **A) Je suis heureux.** ✓ CORRECT - This uses the correct present tense conjugation of être for je.
- **B) Je es heureux.** ✗ The conjugation 'es' is for tu, not je.
- **C) Je est heureux.** ✗ The conjugation 'est' is for il/elle, not je.
- **D) J'étais heureux.** ✗ This uses the imperfect tense, not present tense.

**Question 2**: "You are tired."
- **Hint**: Use the present tense form of être for tu
- **A) Tu es fatigué.** ✓ CORRECT - This is the correct present tense conjugation of être for tu.
- **B) Tu suis fatigué.** ✗ The conjugation 'suis' is for je, not tu.
- **C) Tu êtes fatigué.** ✗ The conjugation 'êtes' is for vous, not tu.
- **D) Tu sera fatigué.** ✗ This uses the future tense, not present tense.

**Question 3**: "He is a student."
- **Hint**: Use the present tense form of être for il
- **A) Il est étudiant.** ✓ CORRECT - This is the correct present tense conjugation of être for il.
- **B) Il es étudiant.** ✗ The conjugation 'es' is for tu, not il.
- **C) Il suis étudiant.** ✗ The conjugation 'suis' is for je, not il.
- **D) Il était étudiant.** ✗ This uses the imperfect tense, not present tense.

**Question 4**: "We are friends."
- **Hint**: Use the present tense form of être for nous
- **A) Nous sommes amis.** ✓ CORRECT - This is the correct present tense conjugation of être for nous.
- **B) Nous sont amis.** ✗ The conjugation 'sont' is for ils/elles, not nous.
- **C) Nous êtes amis.** ✗ The conjugation 'êtes' is for vous, not nous.
- **D) Nous avons amis.** ✗ This uses the wrong verb 'avoir' instead of être.

**Question 5**: "You (plural) are ready."
- **Hint**: Use the present tense form of être for vous
- **A) Vous êtes prêts.** ✓ CORRECT - This is the correct present tense conjugation of être for vous.
- **B) Vous es prêts.** ✗ The conjugation 'es' is for tu, not vous.
- **C) Vous sommes prêts.** ✗ The conjugation 'sommes' is for nous, not vous.
- **D) Vous serez prêts.** ✗ This uses the future tense, not present tense.

**Question 6**: "She is not here."
- **Hint**: Use the negative form with ne...pas around être
- **A) Elle n'est pas ici.** ✓ CORRECT - This correctly places ne...pas around the conjugated verb.
- **B) Elle ne pas est ici.** ✗ The negation is incorrectly placed before the verb.
- **C) Elle est ne pas ici.** ✗ The negation should wrap around the verb, not come after.
- **D) Elle n'était pas ici.** ✗ This uses the imperfect tense instead of present tense.

**Question 7**: "They are busy."
- **Hint**: Use the present tense form of être for ils
- **A) Ils sont occupés.** ✓ CORRECT - This is the correct present tense conjugation of être for ils.
- **B) Ils es occupés.** ✗ The conjugation 'es' is for tu, not ils.
- **C) Ils sommes occupés.** ✗ The conjugation 'sommes' is for nous, not ils.
- **D) Ils étaient occupés.** ✗ This uses the imperfect tense, not present tense.

**Question 8**: "She is a teacher."
- **Hint**: Use the present tense form of être for elle
- **A) Elle est professeure.** ✓ CORRECT - This is the correct present tense conjugation of être for elle.
- **B) Elle suis professeure.** ✗ The conjugation 'suis' is for je, not elle.
- **C) Elle sont professeure.** ✗ The conjugation 'sont' is for ils/elles plural, not elle singular.
- **D) Elle sera professeure.** ✗ This uses the future tense, not present tense.

## COMMON GRAMMAR PATTERNS FOR VERIFICATION

### NEGATION PATTERNS (ne...pas)
- **Présent**: Je ne suis pas → Je ne suis pas heureux
- **Passé Composé**: Je n'ai pas été → Je n'ai pas été malade  
- **Futur Simple**: Je ne serai pas → Je ne serai pas là

### GENDER AGREEMENT
- **Masculine**: heureux, fatigué, occupé, prêt
- **Feminine**: heureuse, fatiguée, occupée, prête
- **Note**: Agreement with subject in être constructions

### PRONUNCIATION HINTS
- **être**: Silent final 'e' in most forms
- **avoir**: Liaison with vowel sounds (j'ai, ils ont)
- **faire**: 'ai' sound in je fais, tu fais, nous faisons
